# WhatStats Features (Concise)

Short, user-facing feature descriptions. Each section maps to a folder in `public/features/` that contains the related images.

## Import WhatsApp Export (import-whatsapp-export)
- Import WhatsApp chat exports into WhatStats.
- Supports ZIP exports and automatic extraction.

## Onboarding Tutorial (onboarding-tutorial)
- Step-by-step walkthrough of key features and how to export chats.

## Chat List (chat-list-management)
- Home screen list of imported chats with access to each analysis.

## Overview Dashboard (overview-dashboard)
- High-level summary of a chat: total messages, participants, and quick stats.

## Day Analysis (day-analysis)
- Hour-by-hour activity breakdown.

## Week Analysis (week-analysis)
- Weekday activity patterns with detail views.

## Emoji Overview (emoji-overview)
- Top emojis and overall emoji usage for the chat.

## Emoji Per-Person (emoji-per-person)
- Emoji usage summaries for each participant.

## Emoji Deep Dive (emoji-deep-dive)
- Detailed emoji stats and personality insights for a participant.

## Chat Wrapped Story (chat-wrapped-story)
- Story-style presentation of chat highlights across multiple cards.

## Share as Image (share-image)
- Generate a shareable image of key chat stats.
